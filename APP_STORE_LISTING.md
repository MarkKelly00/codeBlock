## App Details (500 chars max)

```
Sale Discount Lock helps merchants protect profit margins during sales. When you're running a sitewide sale with automatic discounts, you don't want customers stacking additional discount codes.

With one click in the Checkout Editor, enable Sale Mode to:
• Automatically remove any applied discount codes
• Keep gift cards fully functional
• Show customers a customizable message explaining the sale

No code changes needed. Toggle on/off instantly. Perfect for BFCM, flash sales, and seasonal promotions.
```

## App Review Instructions

```
Testing Sale Discount Lock:

1. Add any product to cart
2. Go to checkout
3. Try applying a discount code (e.g., "SAVE10")
4. Observe: The code is automatically removed
5. See the banner message explaining discounts are disabled
6. Try applying a gift card - it works normally

To toggle Sale Mode:
1. Go to Settings → Checkout → Customize
2. Find "Sale Discount Lock" in the Apps section
3. Toggle "Enable Sale Mode" on/off
4. Save changes

No external accounts required.
Performance impact: Minimal (< 50ms)
```
